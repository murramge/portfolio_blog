## 프로젝트 개요
학습 데이터 관리자가 시험 통계와 문장 분석 데이터를 한눈에 파악할 수 있도록 반응형 디자인의 대시보드를 구축하여, 효율적인 업무 수행을 지원하는 것을 목표로 하였습니다. 이 프로젝트는 다양한 기기에서 최적의 사용자 경험을 제공하면서, 실시간 데이터 시각화와 정확한 데이터 관리를 통해 전반적인 시스템 성능을 향상시키는 데 중점을 두었습니다.

## 주요 기능 및 역할
### 1. 대시보드 UI 설계 및 구현
- shadcn/ui 적용
    - shadcn/ui 템플릿을 활용하여 기본 구조를 반응형으로 커스터마이징함
    - 다양한 기기에서 최적의 사용자 경험을 제공할 수 있도록 디자인을 최적화

- 시험 통계 및 문장 분석 데이터
    - 시험(시험, 지문, 문단, 문제, 선지)과 문장 분석(문장, 속성, 단어, 원형단어, 문법) 데이터를 날짜별로 비교하여, 어제와 오늘의 상승/하락률을 퍼센티지로 시각화

- 상세 인터랙티브 기능
    - shadcn의 Chart를 커스텀하여, 데이터 흐름을 라인차트로 시각화
    - 커스터마이징된 툴팁을 통해 마우스 호버 시 해당 날짜와 데이터 카운트를 확인할 수 있도록 구현했으며, 기간 범위를 설정하면 선택한 기간의 데이터 흐름이 반영되도록 하여 보다 직관적인 분석이 가능

- 관리자별 데이터 처리량 시각화
    - 각 관리자의 데이터 처리량을 수치화하고 랭킹 시스템 형태로 표현하여, 데이터의 정확한 수치와 흐름을 한눈에 파악할 수 있도록 구성

### 2. 효율적인 데이터 연동 및 성능 최적화

- GraphQL 및 캐싱 처리
    - GraphQL 스키마를 통해 데이터를 가져오고, 기간별 키를 저장한 캐싱 처리를 도입함으로써 불필요한 통신을 줄이고 전반적인 시스템 성능을 최적화했습니다.

## 기술 스택
- 프론트엔드: React, TailwindCSS, shadcn/ui, Typescript
- 데이터 시각화: 커스터마이징된 차트 및 인터랙티브 툴팁
- 데이터 연동: GraphQL, Apollo Client
- 상태 관리 및 성능 최적화: 로컬 캐싱, 실시간 데이터 처리
