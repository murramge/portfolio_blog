## 프로젝트 개요
플레이그라운드는 영어 교육 콘텐츠 제작을 위한 학습 관리자용 페이지로, AI를 활용하여 문장, 단어, 문법 데이터를 자동으로 추출하고 이를 관리할 수 있는 인터페이스를 제공합니다. 이 시스템은 학습 데이터의 생성, 수정, 검증 과정을 효율적으로 통합하여 작업 생산성을 극대화하는 것을 목표로 설계되었습니다.

## 주요 기능 및 역할
### 1. 문장 분석
- 절 분리 프로세스
    - AI 기반으로 문장을 절 단위로 분리하여 분석 정확도를 향상.
    - 절 분리 데이터를 기반으로 영문법 요소를 추출하고, 문장의 번역 기능 추가.
- 문장 순서 조정 기능
    - 사용자가 분석된 문장의 순서를 간단히 변경할 수 있도록 Drag-and-Drop (React-DnD) 기능 구현.
    - 과거 직접 수정하던 비효율적인 작업 방식을 개선.
### 2. 단어 선택
- 자동 단어 토큰화
    - 문장 내 단어를 자동으로 추출하고, 단어의 원형 데이터를 제공.
    - 단어와 문장 데이터를 매핑하여 문맥에 따른 단어의 정확성을 보장.
- 단어 번역 및 시각화
    - 단어를 한글로 번역하고, 단어별 정보(원형 및 번역)를 한눈에 확인 가능한 UI 구현.
### 3. 문법 추출
- 품사 태깅 및 문법 데이터 분리
    - AI 모델을 활용하여 문장을 카테고리로 분류하고, 자동으로 문법 요소를 추천.
    - 필요한 경우 사용자가 직접 문법을 검색하여 데이터 완성도를 높이는 인터페이스 제공.
- 문법 매핑 및 트리 시각화
    - 문법 분석 결과를 문장과 연결하며, 결과 데이터를 트리 형태로 시각화하여 사용자 이해도 증대.
### 4. 프로세스 시각화 및 사용자 경험 강화
- 단계별 진행 상황 시각화
    - 문장 분석, 단어 선택, 문법 추출 과정을 프로그레스바와 작업 완료 상태로 시각화하여 사용자가 현재 진행 상태를 직관적으로 파악 가능.
반응형 디자인 및 재사용 가능한 컴포넌트 구현
    - Radix UI와 TailwindCSS를 사용해 사용자 친화적이고 재사용 가능한 인터페이스 설계.
### 5. 데이터 연동 및 상태 관리
- GraphQL과 Apollo Client를 활용한 데이터 연동
    - Mutation, Query, Subscription을 통해 데이터 생성, 수정, 조회를 효율적으로 처리.
    - 로컬 캐싱 및 쿼리 최적화를 통해 데이터 요청 속도를 30% 이상 개선.
- 클라이언트-서버 간 동기화
    - Drag-and-Drop으로 발생한 UI 변경 사항을 서버에 반영하여 클라이언트와 서버 간의 데이터 일관성을 유지.


## 어려웠던 점과 해결 방안
 ### 1. Drag-and-Drop 구현 중 UI-서버 동기화 문제
#### 문제점
- Drag-and-Drop으로 순서를 변경하면, UI와 서버 간 데이터 불일치가 발생.
#### 해결 방법
- React-DnD를 기반으로 UI에서 순서를 업데이트하고, 데이터 변경 요청을 서버로 전송하여 클라이언트-서버 동기화를 구현.
순서 변경 요청이 실패할 경우, 즉시 롤백하는 예외 처리를 추가하여 안정성 확보.
### 2. AI 자동화와 사용자 입력 데이터 유지
#### 문제점
- 단계별 AI 작업 중 사용자가 입력한 데이터를 지속적으로 유지해야 하며, 클라이언트 상태 관리가 복잡해짐.
#### 해결 방법
- 하나의 워크플로우로 모든 AI 작업을 통합 처리하고, 사용자의 입력 데이터를 클라이언트 메모리에 저장하여 작업 흐름 간 데이터가 소실되지 않도록 함.
### 3. 대량 데이터 처리 최적화
#### 문제점
- 대량의 문장 및 단어 데이터를 실시간으로 검색하고 처리하는 과정에서 성능 저하 발생.
#### 해결 방법
- GraphQL에서 Pagination을 적용하고, Apollo Client의 로컬 캐싱을 활용하여 대량 데이터를 효율적으로 처리.

## 기술 스택
- 프론트엔드: React, Radix UI, TailwindCSS, Typescript
- 데이터 연동: GraphQL, Apollo Client
- 상태 관리 및 데이터 최적화: React-DnD, Zod, 로컬 캐싱
- 코드 품질 관리: ESLint, Prettier
